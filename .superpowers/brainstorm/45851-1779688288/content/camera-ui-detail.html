Element Behavior by State

🏷️ Guidance Pill (top center)
No face → "Position your face in the oval"
Too far → "Move a little closer"
Too close → "Move back a little"
Low light → "Find better lighting 💡"
Tilted → "Look straight ahead"
Perfect → "✓ Perfect! Hold still"
⭕ Face Oval
• Dashed when guiding → Solid when perfect
• Color matches guidance pill state
• Subtle scale pulse when waiting for face
• Corner markers for alignment reference
• Smooth spring animation between colors
🟢 Quality Indicator Dots
Three dots below the oval: Light, Distance, Angle
• Green = good
• Yellow = fixable
• Red = blocking
Shows at-a-glance what's working and what needs fixing
⏱️ Auto-Capture Ring
• Appears only in "Perfect" state
• Green arc fills around shutter button (2s)
• If user moves → ring resets, state changes
• Tap shutter anytime to skip countdown
• Haptic pulse on capture
⚡ Smart Flash Suggestion
• Flash icon pulses when in "Low Light" state
• Tooltip: "Try turning on flash"
• NOT auto-enabled (flash washes out skin tones)
• Just a gentle suggestion

Bonus: Micro-interactions that create "WOW"

🎯 Snap-to-oval: When face enters zone, oval gently "hugs" toward face size with spring physics
🌈 Color breathing: Green state has slow subtle glow pulse (alive, not static)
📳 Haptic language: Light tap on each state change, double-tap on "Perfect" lock, strong pulse on capture
✨ Particle burst: Tiny confetti particles on successful auto-capture (celebration moment)
🎵 Sound design: Optional soft "ding" progression (low→high pitch) as conditions are met one by one
📐 Parallax oval: Oval slightly shifts with device gyroscope for depth feel
